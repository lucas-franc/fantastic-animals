# Fantastic Animals

![Site](./img/github/site-1.png)

## About

Fantastic Animals is a web project developed during a JavaScript course. It is a website that provides information about various animals. The animal information is retrieved from a local API, while the Bitcoin price is obtained from a web API. The data is fetched using the `fetch()` function.

## Getting Started

To run the project, please follow these steps:

1. Download the project to your local machine.
2. Open a terminal or command prompt and navigate to the project directory.
3. Run the command `npm run dev` to start the development server.
4. Once the server is running, you can access the website by opening the provided URL or using a local development server.
5. Feel free to make any desired changes to the project.
